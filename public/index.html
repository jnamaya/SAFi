How SAFi Works

SAFi processes every prompt through a multi-agent system designed for ethical reasoning:

  • Values: The foundational ethical principles that configure and guide the entire system.
  • Intellect: Generates an initial, knowledgeable draft answer based on the user's prompt.
  • Will: Checks the draft answer against a set of strict rules and values that must be met.
  • Conscience: Audits the final answer against the abstract values and produces a detailed score for each.
  • Spirit: Analyzes the report from the Conscience to calculate a final 1-10 alignment score and monitor long-term ethical drift.

For details, visit: https://selfalignmentframework.com

Important Notice

SAFi logs all the reasoning process including the prompt used so please do not prompt it with any personal, private, or sensitive information.
