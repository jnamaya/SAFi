# SAFi Explained: The Intellect

## Core concept
The Intellect is the generative faculty in SAFi. Its job is to produce the first draft of any response or tool call proposal. It is the only faculty in the pipeline that uses a large language model for generation.

## The Intent Air Gap
The Intellect is completely severed from the execution environment. It can only produce intents — proposals for what should happen — not actions. Even if the Intellect hallucinates a destructive command or a policy violation, it cannot execute anything on its own. Every output from the Intellect must pass through the Will before it is acted on or delivered to the user.

This is the Intent Air Gap: the architectural separation between generation and execution. The Intellect can imagine anything, but it cannot do anything without the Will's approval.

## Inputs to the Intellect
The Intellect receives four sources of information at each turn.

### User prompt (x_t)
The message sent by the user at turn t.

### Values profile (V)
The Synderesis-compiled configuration for the active persona. The Intellect uses the worldview, style, and scope statement sections to shape the tone, framing, and boundaries of its draft.

### Memory (M_t)
Two types of memory are provided:
1. Conversation memory: a running summary of the dialogue maintained by a background summarizer.
2. Spirit coaching: a feedback note generated by the Spirit faculty at the end of the previous turn. Example: "Coherence 9/10, drift 0.01. Main improvement area: Justice (score: 0.21)." This coaching closes the feedback loop and allows the agent to self-correct over time.

### Retrieved context
Any documents from RAG knowledge bases, plugin output, or MCP tool results assembled during Phase 1 data gathering.

## Outputs from the Intellect
The Intellect produces two outputs.

1. Draft (a_t): the proposed response or tool call.
2. Rationale (r_t): a short internal explanation of the reasoning behind the draft.

Both outputs are passed to the Will faculty for Phase 3 evaluation. Neither reaches the user until it has cleared the Will and the Conscience.

## Phase 0: before the Intellect is called
Before the Intellect generates anything, Phase 0 screens the incoming prompt against the persona's blacklists and known injection patterns. If the prompt fails this check, the Intellect is never called. This ensures the Intellect never processes adversarial inputs that can be caught structurally before generation, preventing unnecessary LLM exposure to attack content.

## Model agnostic
The Intellect faculty is model-agnostic. The underlying LLM can be GPT, Claude, Gemini, Llama, Groq, Mistral, or DeepSeek. Changing the model does not change the governance pipeline that wraps it.
